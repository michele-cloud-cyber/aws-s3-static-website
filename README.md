# AWS S3 Static Website with CloudFront

## Project Overview

This project demonstrates how to deploy a static website using Amazon S3 and distribute it globally with Amazon CloudFront.

## Architecture

User → CloudFront CDN → Amazon S3 Static Website

## AWS Services Used

* Amazon S3 (Static Website Hosting)
* Amazon CloudFront (Content Delivery Network)

## Features

* Static website hosting
* Public S3 bucket policy
* Global CDN distribution with CloudFront
* Low latency content delivery
